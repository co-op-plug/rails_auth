TheAuth

TheAuth是个处理鉴权的Rails Engine

特性

  • 系列项目说明

  • 使用Rails自带的ActiveModel::SecurePassword模块处理密码;

文件说明

如何使用

###

class ApplicationController < ActionController::Base
  include TheAuthController
end

Include in Model

class User
  include TheAuthUser
end
